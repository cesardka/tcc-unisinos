## Como completar cada entrada

Para cada item da seção 🔴/🟡, depois de encontrar no Scholar/repositório:

1. Copie a referência no formato BibTeX (o Scholar tem botão "Cite → BibTeX").
2. Cole os campos faltantes diretamente em `referencias.bib`, preservando a chave atual.
3. Para autores, use o separador `and` no BibTeX:

   ```bibtex
   author = "Sobrenome, Nome and Sobrenome2, Nome2 and ..."
   ```

4. Para URLs de TCCs/dissertações, adicione:

   ```bibtex
   note = "Dispon{\'i}vel em \url{https://...}. Acesso em: DD mes. AAAA."
   ```
